Deloreannn.
Toutes les commandes se terminent par un point-virgule ';'
Commande Paramètre(s) Exemple Commentaire
$ nom_variable $longueurCarre; Utilise une variable
Teleporter coord_x, coord_y Teleporter 150, 150; Téléporte le curseur aux coordonnées choisies
Avancer coord_x, coord_y || distance Avancer 150, 150; Avancer 200; Dessine jusqu'aux coordonnées choisies ou bien dessine de x pixels en suivant la direction définies préalablement
Couleur #xxxxxx Couleur #ff00ff; Choisi la couleur du trait
Epaisseur epaisseur Epaisseur 2; Choisi l'épaisseur du trait
Tourner angle || direction Tourner 180; Tourner gauche; Change la direction du dessin (de 0° à 360°, 0° permet de se diriger vers la droite)
Repeter n_fois Repeter 5; Permet de répéter le code suivant l'instruction. Pour délimiter la fin du bloc, utilisez l'insctruction 'Fin Repeter'
Def_origine coord_x,coord_y Def_origine 150,170; Permet de changer l'origine du repère dans lequel on dessine
